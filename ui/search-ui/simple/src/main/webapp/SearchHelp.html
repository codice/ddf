DDF Search

This widget allows users to search metadata and find holdings that have been exposed through the DDF federation

  • Layout
  • Inputs
  • Results List
  • Metacard View

Search Page Layout

The Search Page contains several panels

Classification Banners
A optional Banner along the top and bottom which states the maximum classification of data available from this system
Menu Bar
Includes links to this help page and back to the search page
Query Pane
A panel on the left side of the page which provides a form for entering query parameters
Query Results Table
A panel to the right of the Query Pane for displaying results from the current query in tabular form

Search Page Query Input Form

The Input form allows the user to specify keyword, geospatial, temporal, and type query parameters. It also allows the user to select the sources to search and the number of results to return.

Keyword Search
A text box allowing the user to enter a textual query. This supports the use of (*) wildcards. If blank, the query will contain to contextual component
Temporal Query
Select from any, relative, or absolute. Any results in no temporal restrictions to the query, relative allows the user to query a period from some length of time in the past until now, and absolute allows the user to specify a start and stop date range
Spatial Search
Select from any, point-radius, and bounding box. Any results in no spatial restrictions to the query, point-radius allows the user to specify a lat/lon and radius to search, and a bounding box allows the user to specify an eastern, western, southern and northern boundary to search within
Type Search
Select from any, or a specific type. Any results in no type restrictions to the query, and specific types shows a list of known content types on the federation, and allows the user to select a specific type to search for
Sources
Select from none, all sources, or specific sources. None results in querying only the local provider, all sources results in an enterprise search where all federations are queried, and specific sources allows the user to select which sources are queried
Results per Page
Select the number of results to be returned by a single query
Submit
Submits the search, and refresh with the result
Clear
Removes all user entered data from the query form

Search Page Results Table

The table of results shows the details of the results found, as well as a link to download the product if applicable.

Result Count
Total Number of Results available for this query. If There are more results than the number displayed per page then a page navigation links will appear to the right
Pages
Provides page navigation, which generate queries for requesting additional pages of results
Results List
This provides a preview and links of the results

The following is a list of columns in in the results table

Title
Displays title of the metacard. This will be a link which can clicked to view the metacard in the Metacard View
Source
Displays where the metadata came from, which could be the local provider or a federated source
Location
Displays the WKT Location of the metacard, if available
Time
Shows the Received (Created) and Effective times of the metacard, if available
Thumbnail
Shows the thumbnail of the metacard, if available
Download
A download link to retrieve the product associated with the metacard, when applicable, if available

Metacard View

This view shows more detailed look at a metacard.

Back to Results Button
Returns the view back to the Results Table
Previous & Next
Navigation to page through the metacards one by one
Metacard Table
Provides the list of properties and associated values of the metacard
Metadata
The metadata, when expanded, displays a tree structure representing the metacard's custom metadata
