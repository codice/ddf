=== Available Input Transformers

The following input transformers are available in a standard installation of DDF:

<<_video_input_transformer,Video Input Transformer>>:: Creates Catalog metacards from certain video file types.

<<_pdf_input_transformer,PDF Input Transformer>>:: Translates a PDF document into a Catalog Metacard.

<<{developing-prefix}xml_input_transformer,XML Input Transformer>>:: Translates an XML document into a Catalog Metacard.

<<_pptx_input_transformer,PPTX Input Transformer>>:: Translates Microsoft PowerPoint (OOXML only) documents into Catalog Metacards.

<<_geojson_input_transformer,GeoJSON Input Transformer>>:: Translates GeoJSON into a Catalog metacard.

<<_tika_input_transformer,Tika Input Transformer>>:: Translates Microsoft Word, Microsoft Excel, Microsoft PowerPoint, OpenOffice Writer, and PDF documents into Catalog records.


=== Available Metacard Transformers

The following metacard transformers are available in a standard installation of DDF:

<<_resource_metacard_transformer,Resource Metacard Transformer>>:: Retrieves the resource bytes of a metacard by returning the resource associated with the metacard.

<<_xml_metacard_transformer,XML Metacard Transformer>>:: Translates a metacard into an XML-formatted document.

<<_kml_metacard_transformer,KML Metacard Transformer>>:: Translates a metacard into a KML-formatted document.

<<_metadata_metacard_transformer,Metadata Metacard Transformer>>:: returns the `Metacard.METADATA` attribute when given a metacard.

<<_geojson_metacard_transformer,GeoJSON Metacard Transformer>>:: Translates a metacard into GeoJSON.

<<_kml_style_mapper,KML Style Mapper>>:: Maps a KML Style URL to a metacard based on that metacard's attributes.

<<_thumbnail_metacard_transformer,Thumbnail Metacard Transformer>>:: Retrieves the thumbnail bytes of a Metacard by returning the `Metacard.THUMBNAIL` attribute value.


=== Available Query Response Transformers

The following query response transformers are available in a standard installation of DDF:

<<_csw_query_response_transformer,CSW Query Response Transformer>>:: Transforms a query response into a http://www.opengeospatial.org/standards/cat[CSW-formatted] document.

<<_query_response_transformer_consumer,Query Response Transformer Consumer>>:: Translates a query response into a Catalog Metacard.

<<_xml_query_response_transformer,XML Query Response Transformer>>:: Translates a query response into an XML-formatted document.

<<_geojson_query_response_transformer,GeoJSON Query Response Transformer>>:: Translates a query response into a GeoJSON-formatted document.

<<_kml_query_response_transformer,KML Query Response Transformer>>:: Translates a query response into a KML-formatted document.

<<_atom_query_response_transformer,Atom Query Response Transformer>>:: Transforms a query response into an http://tools.ietf.org/html/rfc4287[Atom 1.0] feed.

== Catalog Plugins

include::_content/_architectures/_plugins/plugins-intro.adoc[leveloffset=+1]

<<{architecture-prefix}pre_authorization_plugins,Pre-Authorization Plugins>>:: Perform any changes needed before security rules are applied.

<<_access_plugins,Access Plugins>>:: Used to build policy information for requests.

<<{architecture-prefix}pre_ingest_plugins,Pre-Ingest Plugins>>:: Perform any changes to a metacard prior to ingest.

<<_post_ingest_plugins,Post-Ingest Plugins>>:: Perform actions after ingest is completed.

<<_post_process_plugins,Post-Process Plugins>>:: Performs additional processing after ingest.

<<_post_query_plugins,Post-Query Plugins>>:: Perform any changes to a response after query completes.

<<_post_federated_query_plugins,Post-Federated-Query Plugins>>:: Perform any changes to a response after federated query completes.

<<_pre_resource_plugins,Pre-Resource Plugins>>:: Perform any changes to a request associated with a metacard prior to download.

<<_post_resource_plugins,Post-Resource Plugins>>:: Perform any changes to a resource after download.

<<_pre_create_storage_plugins,Pre-Create Storage Plugins>>:: Perform any changes before creating a resource.

<<_post_create_storage_plugins,Post-Create Storage Plugins>>:: Perform any changes after creating a resource.

<<_pre_update_storage_plugins,Pre-Update Storage Plugins>>:: Perform any changes before updating a resource.

<<_pre_subscription_plugins,Pre-Subscription Plugins>>:: Perform any changes before creating a subscription.
